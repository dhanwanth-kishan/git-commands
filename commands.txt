-> command for checking status

git config --list

-> command for creating username

git config --global user.name "user_name"

-> command for adding user email

git config --global user.email "email"

-> command for initializing git
 
 git init

-> command for listing all untracked files

git status

-> command for adding all untracked files to staging area

git add -add

-> command to add all staged files to local repository

git commit -m "meassage"


git commands for adding a remote repository(from github) to git

-> create repo in github and copy repo link 
-> terminal- > git remote add "repo-name" "link"


command for listing remote name(if dont know)

-> git remote -v

command for adding changes in local repo to remote repo

-> git push -u remote-name branch-name
-> eg: git push -u luminar master(by default branch-name = master or main)
-> git branch
